# AuthLib React Source

Editable source for the authentication-template gallery.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:5173`.

## Included

- Eight responsive authentication previews
- React source code
- Tailwind CSS styling
- Social login, password, recovery, 2FA, and enterprise UI examples

These interfaces are visual templates. Connect the forms to your own authentication API before production use.
